##  How It Works

1. **Load your video** â†’ supports `.mp4`, `.mov`, `.avi`, etc.  
2. **Pick a reference frame** â†’ usually frame `0`.  
3. **Annotate objects**:  
   - Left-click = positive point  
   - Right-click = negative point  
   - Press **C** to name the object (must contain `"target"` for gaze markers)  
   - Press **T** to test masks  
   - Press **Enter** when done  
4. **Set detection threshold** â†’ default is 10% overlap.  
5. **Process video** â†’ masks are propagated, overlaps are detected, and outputs are generated.
